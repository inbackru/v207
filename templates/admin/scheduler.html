Описание задач

Напоминания о задачах — проверяет каждые 5 минут, есть ли задачи, которые наступят через 30 минут. Создаёт уведомление менеджеру.
Просроченные задачи — проверяет каждые 10 минут, есть ли просроченные задачи. Создаёт уведомление менеджеру (не чаще 1 раза в 24 часа на задачу).
Моментальные оповещения — проверяет каждые 5 минут новые объекты и отправляет мгновенные уведомления подписчикам.
Ежедневный дайджест — отправляет ежедневную подборку новых объектов в 8:00 утра.
Еженедельный дайджест — отправляет еженедельную подборку каждый понедельник в 8:00 утра.
История цен — снимает ценовой снимок для всех ЖК и квартир 1-го числа каждого месяца. Можно запустить вручную для немедленного снимка.
Автогеокодинг — каждые 2 часа находит ЖК и квартиры без координат и запрашивает их через Яндекс.Геокодер.
Расстояния до центра — пересчитывает distance_to_center для всех ЖК с координатами по формуле Гаверсинуса. Запускается ежедневно в 07:00, после геокодинга.
Инфраструктура районов — обновляет данные инфраструктуры (школы, аптеки, парки и т.д.) для всех районов через Overpass API. Еженедельно в понедельник в 04:00.
Обогащение ЦИАН — полный парсинг ЖК и застройщиков с ЦИАН каждое воскресенье в 02:00. Ежедневно (Пн–Сб) в 06:00 запускается лёгкое обновление только цен.
POI рядом с ЖК — обновляет список объектов инфраструктуры (транспорт, магазины, школы, больницы и т.д.) вблизи каждого жилого комплекса через Overpass API. Еженедельно в среду в 03:00. При первом запуске обрабатывает все ЖК (~8–15 мин), при повторных — обновляет данные.
Парсинг акций из Telegram — читает горячие акции застройщиков из Telegram-группы с ветками. Сопоставляет ветки с ЖК/застройщиками в БД по названию. Сохраняет фото и текст в таблицу telegram_promotions. Требует: TELEGRAM_API_ID, TELEGRAM_API_HASH и сессию sessions/promo_parser.session. Ежедневно в 07:00.
Очистка пропавших объектов — находит квартиры, которые исчезли с ЦИАН (не обновлялись 7+ дней при наличии свежих объектов в том же ЖК), и помечает их как is_active=False, status=sold. Работает только для объектов с inner_id (пришли из парсера). Ежедневно в 05:00.
