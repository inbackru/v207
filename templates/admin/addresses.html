🏠 Адреса ЖК и квартир

Обогащение через DaData, синхронизация адресов с корпусами и квартирами.

🔍 Шаг 1 — DaData: обогатить ЖК (район и микрорайон)

Для каждого ЖК определяет address_city_district (Прикубанский, Адлерский…) и address_quarter (мкр Горхутор, Кудепста…) через DaData /clean + /geolocate. Скорость ~1 ЖК/сек — для 272 ЖК Краснодара займёт ~5 мин. 


  


  
  

    

      📐 Шаг 1б — Bbox-назначение округов (Краснодар)
    

    

      Быстро назначает Прикубанский / Карасунский / Западный / Центральный округ ЖК Краснодара,
      у которых после DaData остался пустой район — по координатам. Не перезаписывает уже заполненные поля.
    

    

      
      
    

    

  


  
  

    

      🔄 Шаг 2 — Синхронизировать адреса в квартиры
    

    

      Копирует address_city_district → parsed_district и address_quarter → parsed_settlement
      из ЖК во все квартиры этого ЖК. Также заполняет address квартир из таблицы корпусов.
