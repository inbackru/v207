Порядок работы

  1. Выберите город в «Автоимпорт из OSM» → нажмите Импортировать из OSM (Районы)
  2. Дождитесь списка импортированных районов (1–2 мин)
  3. Выберите тот же город в PIP-привязке → Привязать к районам
  4. При необходимости — повторите для режима Улицы
  5. SEO-страницы районов добавляются отдельно через код
