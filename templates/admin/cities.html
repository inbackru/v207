Как масштабироваться: регион с множеством городов

Краснодарский край — как устроить

У ЦИАН каждый город — отдельный Region ID. Краснодарский край как регион тоже имеет свой ID — он охватывает все города сразу, но даёт смешанные данные без точного деления.

Правильная стратегия: добавить каждый город отдельно — Краснодар, Сочи, Анапа, Новороссийск и т.д. Каждый получит свой city_id в БД, свои страницы парсинга, и запускается независимо.

Порядок масштабирования

  1. Нажмите «Добавить город» → укажите название и CIAN ID
  2. Нажмите «Проверить» — узнаете сколько объявлений и рекомендуемое кол-во страниц
  3. Создайте город — он появится в списке и в city_config.json
  4. Запустите парсер через «Обогащение данных» — он автоматически обработает новый город
  5. Повторите для следующего города

Как найти CIAN Region ID для любого города

  1. Откройте cian.ru/novostrojki
  2. Нажмите на название города (Москва/регион) в верхней панели
  3. Выберите нужный город и перейдите в раздел новостроек
  4. В URL появится параметр region=XXXXX — это и есть Region ID
  5. Или нажмите «Проверить» рядом с городом ниже — кнопка сама спросит ID и покажет кол-во объявлений
